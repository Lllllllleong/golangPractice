# learningGo

## Overview
learningGo is a simple Go application designed to help you learn the basics of Go programming. This project serves as a starting point for building Go applications and understanding the Go ecosystem.

## Getting Started

### Prerequisites
- Go 1.16 or later installed on your machine.
- A code editor (e.g., Visual Studio Code) with Go support.

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/learningGo.git
   ```
2. Navigate to the project directory:
   ```
   cd learningGo
   ```
3. Install the dependencies:
   ```
   go mod tidy
   ```

### Running the Application
To run the application, use the following command:
```
go run src/main.go
```

### Project Structure
- `src/main.go`: The entry point of the application.
- `go.mod`: Module definition and dependencies.
